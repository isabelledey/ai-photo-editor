# ✨ AI Face Enhancer: High-Resolution Facial Refinement

**AI Face Enhancer** is a sophisticated web tool that leverages generative AI models to transform low-quality portraits into high-resolution, professional-grade images. By integrating cutting-edge diffusion models with a modern web interface, it provides seamless facial restoration and intelligent upscaling.

---

## 🚀 Key Features

- **Generative Facial Restoration**: Reconstructs facial details and textures using advanced generative AI.
- **High-Definition Upscaling**: Enhances low-resolution images while maintaining natural-looking skin textures and features.
- **Aesthetic UI**: A premium, developer-centric interface designed with **v0.dev** and customized for a *Clean Beauty* high-end aesthetic.
- **Real-time Processing**: Fast, asynchronous image processing handled through cloud-based AI endpoints.

## 🛠️ Tech Stack

- **Frontend**: React (Vite), Tailwind **CSS**, Framer Motion.
- **UI Scaffolding**: v0.dev for rapid, high-fidelity frontend components.
- **AI/ML Integration**: Utilizes generative models via **Replicate** or **Gemini **API**** for sophisticated facial manipulation.
- **Workflow**: Developed within an Apple Silicon (M1) environment and deployed via **Vercel/Netlify**.

## 📦 Installation & Setup

To run the AI Face Enhancer locally, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone [https://github.com/isabelledey/ai-face-enhancer.git](https://github.com/isabelledey/ai-face-enhancer.git)
    cd ai-face-enhancer
Set up Environment Variables:

Create a .env file in the root directory and add your **API** keys:

    ```bash
    VITE_REPLICATE_API_TOKEN=your_token_here
    VITE_AI_MODEL_ENDPOINT=your_model_url

Start the development server:

    ```bash
    npm run dev
